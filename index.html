C# 泛型

什么是泛型？

面对象编程有一个重要的好处是“代码重用”，它可以极大提高开发效率，代码重用一般是派生出一个类，让它继承基类的所有能力，派生类只需重写虚方法或添加一些新方法就可以定制派生类的功能。泛型是CLR和编程语言提供的一种特殊机制，它支持另一种形式的代码重用——“算法重用”，通过使用泛型，我们可以极大地提高代码的重用度，同时还可以获得强类型的支持，避免了隐式的装箱、拆箱，在一定程度上提升了应用程序的性能。

我们经常会定义一些算法，比如排序，搜索，交换，比较或转换等，但是往往定义算法的人并不知道该算法要操作的对象什么数据类型，所以我们想能不能有一种方法可以使我们的算法广泛地应用到不同类型呢，泛型就很好地解决了这个问题。

Oracle DBA 数据字典整理

整理下 Oracle 常用的数据字典

C# 构造函数学习

C# 构造函数：

new一个类或结构时会调用构造函数，其目的是为了给初始化的对象赋值；类或结构可能具有多个带有不同参数的构造函数，使用构造函数，就可以设置默认值，限制实例化，使代码更加灵活易读。

默认构造函数：

每个 C# 类都具有默认的构造函数，如果没有为类提供构造函数， C# 默认会 创建一个会实例化对象并将成员变量设置为默认值的构造函数， 如果没有为结构提供构造函数，C# 将依赖于隐式默认构造函数，自动将值类型的每个字段初始化为其默认值。
