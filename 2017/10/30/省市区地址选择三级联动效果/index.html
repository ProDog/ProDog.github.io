省市区地址选择三级联动效果

C#
  • C#

介绍

经常会遇到填写地址的需求，地址选择比较常用的一种方式就是省市区三级联动，选择省份，市区的选择项根据省份自动匹配，十分方便，查找资料发现实现方式也有多种，比如将行政区划存入数据库中，取出数据再绑定到选择控件上；以各种格式将行政区划存储在服务端或本地，读取文件解析再绑定到选择控件；还有调用相关接口实现等。今天来研究一下，选择下载行政区划 xml 文件，在程序中解析然后绑定到控件上来实现。

行政区资源

最权威的行政区划来源当然是 国家统计局的数据了，接下来要写个爬虫爬取最新行政区划信息，先留个坑，本次使用前人整理好的数据，xml 格式，可以从 这里 下载。
下载的文件中有省市区的 xml 记录，程序中解析该文件绑定值到控件上实现联动效果。
