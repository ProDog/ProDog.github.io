常见问题

最近使用 Oracle 遇到一个报错，信息如下

异常提示:Oracle数据执行异常，请联系管理员处理
异常信息:ORA-14450: 试图访问已经在使用的事务处理临时表

原因分析

查找相关资料，发现报错原因如下：
在存储过程中使用了 alter 等语句操作临时表，当两个用户(会话)同时执行该处时就会报错，因为执行语句时会给临时表添加 type 为 ‘TO’ 的 Temporary Object 锁，可以理解为临时表操作锁，此时第一个会话如果未正确结束，锁得不到释放，后面的会话处理时就会报错，解决方法可以是手动清掉临时表锁：
